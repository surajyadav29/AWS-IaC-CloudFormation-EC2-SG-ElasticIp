# Project Title

AWS Infrastructure Deployment using CloudFormation (EC2 + S3 + Elastic IP)

# Overview
This project demonstrates how to provision and manage AWS infrastructure using Infrastructure as Code (IaC) with AWS CloudFormation.

The stack automatically creates:
EC2 instances
S3 bucket
Elastic IP for static access

# Tech Stack
AWS CloudFormation
Amazon EC2
Amazon S3
Elastic IP
YAML

# Features
Infrastructure provisioning using CloudFormation templates
Automated EC2 instance deployment
Static IP allocation using Elastic IP
Secure and scalable S3 storage setup
Reusable and parameterized templates

# Architecture
EC2 instance connected to internet via Elastic IP
S3 bucket for storage
CloudFormation orchestrates all resources
